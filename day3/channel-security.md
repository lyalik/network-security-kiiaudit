Передача данных АСУ ТП между филиалами водоканала (например, насосные станции ↔ центральный диспетчерский пункт) требует шифрования по ГОСТ Р 34.12-2015 («Кузнечик») согласно мере ЗИС.19 ФСТЭК №235. Аутентификация участников канала связи выполняется по ГОСТ Р 34.10-2012 (эллиптические кривые). Использование импортных алгоритмов (AES, RSA) без ГОСТ запрещено для КИИ — это нарушение ст. 17 187-ФЗ. Реализация: С-Терра Шлюз на границе каждого филиала с настройкой криптографических туннелей по стандарту ТК26.

# Защита уровня L2 для КИИ

Атаки на коммутаторы (подмена корневого моста, фальшивый DHCP) позволяют злоумышленнику перехватить трафик АСУ ТП без прохождения С-Терра.

Ключевые меры:
- BPDU Guard — отключает порт при атаке на STP
- DHCP Snooping + DAI — блокируют подмену шлюза и ARP-spoofing
- Port Security — ограничивает подключение несанкционированных устройств

Требование ФСТЭК №235: защита от несанкционированного подключения к сети КИИ.

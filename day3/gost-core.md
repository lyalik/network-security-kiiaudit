# ГОСТ-криптография для КИИ

Для защиты каналов связи КИИ требуется шифрование по ГОСТ Р 34.12-2015 («Кузнечик») и аутентификация по ГОСТ Р 34.10-2012.

Требования ФСТЭК №235:
- ЗИС.19: шифрование трафика между филиалами КИИ
- ЗИС.20: аутентификация всех участников канала связи

С-Терра Шлюз обеспечивает шифрование по ГОСТ на границе сети. Для АСУ ТП применяется С-Терра Юнит — лёгкий агент без влияния на работу контроллеров.

Передача данных АСУ ТП без ГОСТ-шифрования = нарушение 187-ФЗ ст. 17.

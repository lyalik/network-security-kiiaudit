# Threat model / Риск-матрица (водоканал КИИ)

## Акторы и цели

- Внутренний нарушитель (ИТ/АСУ ТП): несанкционированный доступ, обход сегментации.
- Внешний нарушитель из интернета: DoS, попытка MITM, проникновение через DMZ.
- Поставщик/подрядчик: ошибки конфигурации, привнесение вредоносного ПО.

## Топ-5 угроз и меры

- MITM/ARP-spoof в L2 → DAI, DHCP Snooping, Port Security, BPDU Guard; мониторинг срабатываний (День 2).
- Rogue DHCP/подмена шлюза → DHCP Snooping + статический default gateway на контроллерах; журналы.
- Прямой доступ из корпсети к АСУ ТП → VLAN/ACL, С-Терра Шлюз, DMZ; проверка deny логов.
- DoS/скан из интернета → Фильтрация на С-Терра Шлюз, rate-limit, белые списки; алерты в SIEM.
- Утечка/подмена телеметрии → ГОСТ VPN (Кузнечик) между филиалами/DMZ и Юнитами; контроль целостности.

## Риски (скоринг упрощённый)

- Высокий: MITM в L2 без DAI/DHCP Snooping → закрыто внедрёнными мерами.
- Средний: Ошибки ключевого управления ГОСТ → требуется регламент ротации/резервного хранения.
- Средний: Недостаток мониторинга логов С-Терра → нужен дашборд/экспорт в SIEM.
- Низкий: Временный доступ подрядчиков → контролируется через ACL+учётные записи с ограниченным временем.

## План контроля и мониторинга

- События С-Терра (ACL deny, DAI, VPN) → SIEM/ГосСОПКА, хранение ≥1 год, ротация.
- Еженедельная проверка: список активных портов, таблицы DHCP Snooping, DAI statistics.
- Ежеквартально: тест на проникновение из корпсети в АСУ ТП и из DMZ в интернет (negative test).
- Инциденты: таймлайн 24 часа, шаблон уведомления, RACI указан в `incident-scenario.md`.

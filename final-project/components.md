# Компоненты архитектуры защиты водоканала

## С-Терра Шлюз
- Расположение: граница интернет ↔ корпоративная сеть
- Функции: 
  • Фильтрация трафика по белым спискам
  • Шифрование межфилиальных каналов по ГОСТ 34.12-2015 «Кузнечик»
  • Интеграция с ГосСОПКА (передача событий безопасности)

## С-Терра Юнит
- Расположение: на контроллерах АСУ ТП (ПЛК насосных станций)
- Функции:
  • Лёгкий агент без влияния на работу промышленных систем
  • Контроль исходящего трафика с контроллеров
  • Передача событий в С-Терра КП/ГСУ

## С-Терра СОВ
- Расположение: мониторинг трафика на границе АСУ ТП
- Функции:
  • Обнаружение аномалий и атак на АСУ ТП
  • Автоматическая блокировка источников атаки

## Требования ФСТЭК №235
- ЗИС.4: сегментация АСУ ТП от корпоративной сети
- ЗИС.5: организация DMZ для серверов мониторинга
- ЗИС.19: шифрование каналов связи по ГОСТ
- ЗИС.20: аутентификация по ГОСТ 34.10-2012

# Аудит КИИ по ГОСТ Р 57580

Главное на аудите ФСТЭК — не «правильная конфигурация», а **документальное подтверждение** соответствия требованиям.

Ключевые документы для проверки:
- Журнал инцидентов (хранение ≥ 12 месяцев, 187-ФЗ ст. 14)
- Перечень СЗИ с сертификатами ФСТЭК/ФСБ
- Схема сегментации сети с границами АСУ ТП
- Акты проверки уязвимостей (не реже 1 раза в квартал)

Отсутствие документа = замечание, даже если технически всё настроено правильно.

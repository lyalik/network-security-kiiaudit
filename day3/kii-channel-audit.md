# Аудит каналов связи КИИ по требованиям ФСТЭК №235

## Требования к шифрованию (ЗИС.19)
Передача данных АСУ ТП между филиалами КИИ ОБЯЗАНА шифроваться по ГОСТ Р 34.12-2015 «Кузнечик» (128-бит). Использование «Магма» (64-бит) или импортных алгоритмов (AES) для АСУ ТП запрещено.

## Требования к аутентификации (ЗИС.20)
Участники канала связи (С-Терра Шлюз филиала ↔ С-Терра Шлюз центра) должны аутентифицироваться по ГОСТ Р 34.10-2012 (эллиптические кривые). Аутентификация по паролю или сертификатам с RSA недопустима.

## Критические ошибки при аудите
1. Каналы АСУ ТП без шифрования («прозрачный» трафик)
2. Использование AES/RSA вместо ГОСТ
3. Отсутствие аутентификации по ГОСТ 34.10-2012
4. Шифрование «Магма» (64-бит) для критичных систем

# Архитектура защиты водоканала как объекта КИИ

## Структура сети

Интернет
    │
    ▼
[С-Терра Шлюз] ← граница КИИ (сертификат ФСТЭК №_____)
    │
    ├── [DMZ: сервер мониторинга]
    │   │   • Доступ из интернета: только порт 443 (HTTPS)
    │   │   • Доступ в АСУ ТП: только порт 502 (Modbus)
    │   │
    │   ▼
    │ [С-Терра Юнит] ← лёгкий агент на контроллерах
    │   │
    │   ▼
    │ [АСУ ТП: насосы, КИПиА] ← ИЗОЛИРОВАННЫЙ СЕГМЕНТ
    │   • Запрещён прямой доступ из корпоративной сети
    │   • Только через С-Терра Юнит из DMZ
    │
    └── [Корпоративная сеть: офисные ПК]
        • Доступ в интернет: разрешён
        • Доступ в АСУ ТП: ЗАПРЕЩЁН (политика С-Терра Шлюз)

## Соответствие требованиям

✅ Постановление №1119 п. 14 — сегментация АСУ ТП от корпоративной сети  
✅ ФСТЭК №235 ЗИС.4 — изоляция критичных систем  
✅ ФСТЭК №235 ЗИС.5 — организация DMZ для серверов мониторинга  
✅ 187-ФЗ ст. 18 — готовность к уведомлению ФСБ в течение 24 часов
